# Universal RPG Platform

This directory is a game-agnostic RPG platform. Any tabletop RPG can be run here once its source books are ingested. Two permanent skills handle orchestration; all game-specific skills are generated automatically during ingestion.

## Skills

### Permanent (work for any game)

| Skill | Usage | Purpose |
|-------|-------|---------|
| `/play` | `/play <game name>` | Start or continue any game. Routes to ingest if needed, then character selection and storyteller. |
| `/rpg-ingest` | `/rpg-ingest <slug> "<Game Name>"` | Ingest PDF source books for a game. Generates 4 game-specific skills. |

### Game-Specific (generated by `/rpg-ingest`)

After ingesting a game with slug `SLUG`, these skills are created:

| Skill | Purpose |
|-------|---------|
| `/SLUG-character` | Create characters — **only for that game** |
| `/SLUG-npc` | Generate NPCs — **only for that game** |
| `/SLUG-combat` | Run combat — **only for that game** |
| `/SLUG-advance` | Spend XP / advance traits — **only for that game** |

Each generated skill has `game:` and `game_name:` frontmatter to prevent cross-game use.

## Directory Layout

```
books/                        ← Drop PDFs here before ingesting
RPG/
  [game_slug]/
    _meta.json                ← Game name, books used, date ingested
    rules/
      char_creation.md        ← Attribute lists, point allocation, creation steps
      combat.md               ← Initiative, actions, damage, conditions
      advancement.md          ← XP costs, prerequisites, maximums
      systems.md              ← Core dice mechanic, modifiers, resource spending
    lore/
      world.md                ← Setting, factions, cosmology
      tone.md                 ← Genre, atmosphere, narrative style
    characters/
      [name_slug].json        ← Character sheet (updated live during play)
    npcs/
      [name_slug].json        ← NPC profile + interaction history
    sessions/
      [char_slug]_YYYY-MM-DD.md  ← Session log, scene-by-scene

scripts/
  extract_pdf_text.py         ← Generic PDF → page_NNNN.txt extractor
  extract_wod_pdfs.py         ← Column-detection library used by extract_pdf_text.py
  generate_image.py           ← Pollinations.ai image downloader
```

## Quickstart

### Play any game
```
/play <game name>
```
- If the game isn't set up yet: scans `books/`, asks which PDFs to use, ingests them, generates skills
- If the game is set up: shows character list and session history

### Add a new game from scratch
1. Copy source PDFs to `books/`
2. `/play <game name>` — it will detect the game is unknown and guide you through ingest

### Manually ingest books
```
/rpg-ingest <slug> "<Game Name>"
```

## Player Controls During a Session

| Input | Effect |
|-------|--------|
| `{action in curly braces}` | In-game character action |
| `"speech in quotes"` | Character dialogue |
| `<angle brackets>` | Out-of-character question or note |
| `<advance X>` | Spend XP to raise trait X |
| `<award N xp>` | Storyteller awards N XP |
| `<save>` | Force an immediate session log checkpoint |
| `<quit>` | End session, save log and character sheet |

## Storyteller Format

Every Storyteller response:
- 1000–3000 characters of vivid narrative
- Ends with exactly **5 numbered choices** in `{curly braces}`
- Dice rolls shown in `(parentheses)` with full breakdown
- OOC notes in `<angle brackets>`

## Notes

- All character JSON files are updated live (atomic write via temp-file rename)
- NPC files are created automatically when new characters appear in the story
- Session logs append scene-by-scene; never overwritten
- XP is awarded per scene and logged in the character sheet
- Game-specific skills are self-contained and must not reference other games' rules
